# Weather App 🌤️
Live weather with 7-day forecast powered by OpenWeatherMap API.
Built with React + Chart.js.

## Features
- Real-time temperature, humidity, wind speed
- 7-day forecast with animated icons
- Search any city worldwide
- Celsius/Fahrenheit toggle

## Setup
```bash
npm install
REACT_APP_WEATHER_KEY=your_key npm start
```
